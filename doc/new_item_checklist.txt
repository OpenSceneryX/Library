Checklist for New Items
=======================

0. Investigate copyright

1. Check object origin (tip of nose for aircraft) on all LOD layers

2. Check object size

3. Check orientation (pointing North) on all LOD layers

4. Convert to v8 format?

5. If there are ground shadows, check they have poly_offset set (in Blender, click the 'Tiles' button)

5. Check any PNGs are RGB, not indexed (X-Plane has a problem rendering indexed PNGs with a background colour as transarent)

6. Check texture size and check powers of 2, scale down if appropriate

7. Run textures through Ping

8. If info.txt has been supplied, check line endings are UNIX

9. Add to release notes

10. Tweet

11. Commit
