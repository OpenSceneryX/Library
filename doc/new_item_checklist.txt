Checklist for New Items
=======================

0. Investigate copyright

1. Check object origin (tip of nose for aircraft) on all LOD layers

2. Check object size

3. Check orientation (pointing North) on all LOD layers

4. Convert to v8 format?

5. Remove any baked-in ground shadows

6. Check any PNGs are RGB, not indexed (X-Plane has a problem rendering indexed PNGs with a background colour as transarent)

7. Check texture size and check powers of 2, scale down if appropriate

8. Run textures through ImageOptim

9. If info.txt has been supplied, check line endings are UNIX

10. Build library

11. Add to release notes

12. Tweet

13. Commit
