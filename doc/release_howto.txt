Release Howto
=============

1. Update release notes to include version and date

2. Check "Enhanced By" logo is at correct version number

3. Clean target folder and build library

4. Create a copy of the built OpenSceneryX-x.x.x folder, rename it to 'OpenSceneryX' and zip this to create the monolithic zip file.

5. Run Manifester on built OpenSceneryX-x.x.x folder and save manifext.xml

6. Run Packager

7. Zip OpenSceneryX-x.x.x.tar, OpenSceneryX-Website-x.x.x.tar and OpenSceneryX-DeveloperPack-x.x.x/

8. Zip manifest.xml

9. Upload zips into /new/

10. Upload manifest.xml.zip into /new/

11. Clear /old

12. Copy /index.html into /old/

13. Extract website zip on site into /new/

14. Extract OpenSceneryX-x.x.x.tar.zip into /new/

15. Copy /index-sitedown.html over /index.html

16. Move /doc/, /facades/, /forests/, /lines/, /manifest.xml.zip, /objects/ and /repository/ into /old/

17. Copy /downloads/ into /old/

18. Move /new/OpenSceneryX-x.x.x/ to /repository/

19. Move /new/OpenSceneryX-Website-x.x.x/* to /

20. Move /new/manifest.xml.zip to /

21. Move /new/OpenSceneryX-DeveloperPack-x.x.x.zip to /downloads/

22. Remove old /new/OpenSceneryX-DeveloperPack-y.y.y.zip from /downloads/

23. chgrp -R www *

24. Ensure /versioninfo has the latest information about the installer in it.

25. Upload the monolithic zip file 'OpenSceneryX.zip' into /downloads/

26. Run 'python sitemap_gen.py --config=config.xml' 

27. Upload sitemap.xml from OpenSceneryX-Website-x.x.x/
