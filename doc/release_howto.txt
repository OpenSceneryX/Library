Release Howto
=============

1. Check the build for missing virtual paths

  $ python check.py

2. Create release branch

3. Update release notes to include version and date

4. Update "Enhanced By" logo to correct version number

5. Commit release changes, tag and merge release changes back into develop

6. Clean target folder and build library

  $ python build.py
  
  ENSURE 'y' is selected for 'Build PDF'

7. Create a copy of the built OpenSceneryX-x.x.x folder, rename it to 'OpenSceneryX' and zip this to create the monolithic zip file:

  $ export COPYFILE_DISABLE=true
  $ zip -r OpenSceneryX.zip OpenSceneryX

8. Run Manifester on built OpenSceneryX-x.x.x folder and save manifext.xml

9. Run Packager

  $ python package.py

10. Zip OpenSceneryX-x.x.x.tar, OpenSceneryX-Website-x.x.x.tar and OpenSceneryX-DeveloperPack-x.x.x/

	$ zip OpenSceneryX-x.x.x.tar.zip OpenSceneryX-x.x.x.tar
	$ zip OpenSceneryX-Website-x.x.x.tar.zip OpenSceneryX-Website-x.x.x.tar
	$ zip -r OpenSceneryX-DeveloperPack-x.x.x.zip OpenSceneryX-DeveloperPack-x.x.x

11. Zip manifest.xml using the command line

	$ zip manifest.xml.zip manifest.xml

12. Clear /new and /old

13. Upload OpenSceneryX-x.x.x.tar.zip, OpenSceneryX-Website-x.x.x.tar.zip and OpenSceneryX-DeveloperPack-x.x.x.zip into /new/

14. Upload manifest.xml.zip and manifest.xml into /new/

15. Copy /index.html into /old/

16. Copy /downloads/ into /old/

----------------------------------------------

17. Extract website zip on site into /new/

18. Extract OpenSceneryX-x.x.x.tar.zip into /new/

19. Copy /index-sitedown.html over /index.html

20. Move /doc/, /extras/, /facades/, /forests/, /lines/, /manifest.xml, /manifest.xml.zip, /objects/, /polygons/, /repository/ and /sitemap.xml into /old/

21. Move /new/OpenSceneryX-x.x.x/ to /repository/

22. Move /new/OpenSceneryX-Website-x.x.x/* to /

23. Move /new/manifest.xml.zip and /new/manifest.xml to /

24. Move /new/OpenSceneryX-DeveloperPack-x.x.x.zip to /downloads/

25. Remove old OpenSceneryX-DeveloperPack-y.y.y.zip from /downloads/

26. Ensure /versioninfo has the latest information about the installer in it.

27. Upload the monolithic zip file 'OpenSceneryX.zip' into /downloads/

28. Log in to Google Webmaster tools and resubmit sitemap

29. Create new section in release note for next changes

30. Promotion
