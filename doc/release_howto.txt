Release Howto
=============

1. Update release notes to include version and date

2. Check "Enhanced By" logo is at correct version number

3. Clean target folder and build library

5. Run 

4. Create a copy of the built OpenSceneryX-x.x.x folder, rename it to 'OpenSceneryX' and zip this to create the monolithic zip file.

5. Run Manifester on built OpenSceneryX-x.x.x folder and save manifext.xml

6. Run Packager

7. Zip OpenSceneryX-x.x.x.tar, OpenSceneryX-Website-x.x.x.tar and OpenSceneryX-DeveloperPack-x.x.x/

8. Upload zips into /new/

9. Upload manifest.xml into /new/

10. Clear /old

11. Copy /index.html into /old/

12. Extract website zip on site into /new/

13. Extract OpenSceneryX-x.x.x.tar.zip into /new/

14. Copy /index_offline.html over /index.html

15. Move /doc/, /facades/, /forests/, /lines/, /manifest.xml, /objects/ and /repository/ into /old/

16. Copy /downloads/ into /old/

17. Move /new/OpenSceneryX-x.x.x/ to /repository/

18. Move /new/OpenSceneryX-Website-x.x.x/* to /

19. Move /new/manifest.xml to /

20. Move /new/OpenSceneryX-DeveloperPack-x.x.x.zip to /downloads/

21. Remove old /new/OpenSceneryX-DeveloperPack-y.y.y.zip from /downloads/

22. chgrp -R www *

23. Ensure /versioninfo has the latest information about the installer in it.

24. Upload the monolithic zip file 'OpenSceneryX.zip' into /new/

25. When upload complete, move /new/OpenSceneryX.zip to /downloads/

26. Run sitemap_gen.py --config=config.xml 

27. Upload sitemap.xml from OpenSceneryX-Website-x.x.x/
