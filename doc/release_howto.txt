Release Howto
=============

1. Update release notes to include version and date

2. Update "Enhanced By" logo to correct version number

3. Commit release changes and tag

4. Clean target folder and build library

  $ python build.py

5. Create a copy of the built OpenSceneryX-x.x.x folder, rename it to 'OpenSceneryX' and zip this to create the monolithic zip file:

  $ export COPYFILE_DISABLE=true
  $ zip -r OpenSceneryX.zip OpenSceneryX

6. Run Manifester on built OpenSceneryX-x.x.x folder and save manifext.xml

7. Run Packager

  $ python package.py

8. Zip OpenSceneryX-x.x.x.tar, OpenSceneryX-Website-x.x.x.tar and OpenSceneryX-DeveloperPack-x.x.x/

9. Zip manifest.xml using the command line

	$ zip manifest.xml.zip manifest.xml

10. Upload zips into /new/

11. Upload manifest.xml.zip into /new/

12. Clear /old

13. Copy /index.html into /old/

14. Copy /downloads/ into /old/

15. Extract website zip on site into /new/

16. Extract OpenSceneryX-x.x.x.tar.zip into /new/

17. Copy /index-sitedown.html over /index.html

18. Move /doc/, /extras/, /facades/, /forests/, /lines/, /manifest.xml.zip, /objects/ and /repository/ into /old/

19. Move /new/OpenSceneryX-x.x.x/ to /repository/

20. Move /new/OpenSceneryX-Website-x.x.x/* to /

22. Move /new/manifest.xml.zip to /

22. Move /new/OpenSceneryX-DeveloperPack-x.x.x.zip to /downloads/

23. Remove old OpenSceneryX-DeveloperPack-y.y.y.zip from /downloads/

24. Ensure /versioninfo has the latest information about the installer in it.

25. Upload the monolithic zip file 'OpenSceneryX.zip' into /downloads/

26. Go to http://www.neuroticweb.com/recursos/sitemap/ and generate a sitemap (Deprecated: run 'python sitemap_gen.py --config=config.xml')

27. Upload sitemap.xml from OpenSceneryX-Website-x.x.x/, log in to Webmaster tools and resubmit sitemap

28. Create new section in release note for next changes

29. Promotion
