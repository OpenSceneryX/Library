Release Howto
=============

0. Pre-build the library

  $ python build.py

1. Check the build for missing virtual paths

  $ python check.py

2. Create release branch

3. Update release notes to include version and date

4. Update "Enhanced By" logo to correct version number

5. Commit release changes, tag and merge release changes back into develop

6. Clean target folder and build library

  $ python build.py
  
  ENSURE 'y' is selected for 'Build PDF'

7. Create a copy of the built OpenSceneryX-x.x.x folder, rename it to 'OpenSceneryX'

8. Zip this to create the monolithic zip file:

  $ export COPYFILE_DISABLE=true
  $ zip -r OpenSceneryX.zip OpenSceneryX

9. Run Manifester on built OpenSceneryX-x.x.x folder and save manifest.xml

10. Run Packager

  $ python package.py

11. Zip OpenSceneryX-x.x.x.tar, OpenSceneryX-Website-x.x.x.tar and OpenSceneryX-DeveloperPack-x.x.x/

	$ zip OpenSceneryX-x.x.x.tar.zip OpenSceneryX-x.x.x.tar
	$ zip OpenSceneryX-Website-x.x.x.tar.zip OpenSceneryX-Website-x.x.x.tar
	$ zip -r OpenSceneryX-DeveloperPack-x.x.x.zip OpenSceneryX-DeveloperPack-x.x.x

12. Zip manifest.xml using the command line

	$ zip manifest.xml.zip manifest.xml

13. Clear /new and /old on the server

14. Upload OpenSceneryX-x.x.x.tar.zip, OpenSceneryX-Website-x.x.x.tar.zip and OpenSceneryX-DeveloperPack-x.x.x.zip into /new/

15. Upload manifest.xml.zip and manifest.xml into /new/

16. Copy /index.html into /old/

17. Copy /downloads/ into /old/

18. Prepare blog post while uploads ongoing

----------------------------------------------

19. Extract website zip on site into /new/

20. Extract OpenSceneryX-x.x.x.tar.zip into /new/

21. Copy /index-sitedown.html over /index.html

22. Move /doc/, /extras/, /facades/, /forests/, /lines/, /manifest.xml, /manifest.xml.zip, /objects/, /polygons/, /repository/ and /sitemap.xml into /old/

23. Move /new/OpenSceneryX-x.x.x/ to /repository/

24. Move /new/OpenSceneryX-Website-x.x.x/* to /

25. Move /new/manifest.xml.zip and /new/manifest.xml to /

26. Move /new/OpenSceneryX-DeveloperPack-x.x.x.zip to /downloads/

27. Remove old OpenSceneryX-DeveloperPack-y.y.y.zip from /downloads/

28. Ensure /versioninfo has the latest information about the installer in it.

29. Correct ownership of folder structure

  $ chown -R apache:apache public_html
  
30. Log into Cloudflare and clear all caches.

31. Upload the monolithic zip file 'OpenSceneryX.zip' into /downloads/

32. Log in to Google Webmaster tools and resubmit sitemap

33. Create new section in release note for next changes

34. Promotion:
	- Update x-plane.org Download Manager entry
	- Update first post in x-plane.org main OpenSceneryX thread
	- Create a new post in the x-plane.org 'News' forum
	- Post an article in the blog
	- Tweet
